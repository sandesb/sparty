# Sparkonto Site

Standalone SPAR/Sparkonto web app extracted from the previous `/spar` route.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Create a new repository for this app

From inside `sparkonto-site/`:

```bash
git init
git add .
git commit -m "Initial Sparkonto standalone site"
git branch -M main
```

Then create/push to your new remote repository of choice.
